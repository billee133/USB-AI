# USB-AI 操作手册 v4.5.7

## 一、快速开始

### Windows
1. 解压到任意目录（含 U 盘）
2. 双击 `启动服务器模式.bat`
3. 浏览器自动打开 `http://localhost:8082`
4. 点击右上角 ⚙ → 填入 DeepSeek API Key → 保存

### 首次使用
- 注册 DeepSeek: https://platform.deepseek.com
- 获取 API Key（sk- 开头）
- 在设置中填入并保存

### 本地模型安装（llama-cpp-python）
1. 运行 `setup_local_model.bat`（自动装 llama-cpp-python + 下载小模型）
2. 重启 USB-AI 服务器
3. 设置中自动出现 GGUF 文件选项
4. 卸载运行 `uninstall_local_model.bat`，零残留回退到 Ollama/API

## 四、联网搜索

### 智能触发（推荐）
默认关闭。当问题包含实时关键词时**自动搜索**：
- 财经：黄金/白银/比特币/汇率/石油/期货
- 天气：气温/台风/雾霾
- 彩票：快乐8/大乐透/双色球
- 体育：比分/NBA/英超/欧冠
- 商品：价格/性价比/多少钱
- 汽车：买车/报价/配置/保养
- 手机：iPhone/华为/小米/回收
- 考试：高考/中考/真题/试题/作文/分数线
- 新闻：新闻/热点/快讯
- AI：GPT/Claude/大模型

### 手动搜索
输入 `/search 关键词` → 显示搜索结果 → 继续追问 AI 会基于结果回答

### 强制开关
🌐 按钮打开 → 所有问题都搜索；关闭 → 仅实时关键词触发

## 五、桌面自动化

需安装 `pyautogui + pillow`，设置面板一键操作。

### 启用方法
1. 确保在服务器模式（双击 `启动服务器模式.bat`）
2. ⚙ 设置 → 勾选「🤖 桌面自动化」
3. 首次开启会弹出确认安装框，点确定自动执行 `pip install pyautogui pillow`
4. 安装完成后自动开启

### 可用功能
| 功能 | 说明 | 对话触发 |
|------|------|----------|
| 截屏 | 截取当前屏幕，返回图片 | "截个屏" |
| 鼠标点击 | 模拟鼠标点击指定坐标 | — |
| 键盘输入 | 在当前光标位置打字 | "输入你好" |
| 快捷键 | 发送组合键（如 Ctrl+C） | "按保存快捷键" |
| 打开浏览器 | 打开指定 URL（仅 http/https） | "打开百度" |
| 定时任务 | 每 N 秒执行一次动作 | "每30秒截屏" |

### 安全说明
- 鼠标/键盘/快捷键操作会弹出确认框
- 浏览器只允许 http/https 协议
- Ctrl+Alt+Del 和 Win+R 被硬阻断
- 所有操作需本地 token 认证

### 卸载
1. ⚙ 设置 → 取消勾选「🤖 桌面自动化」
2. 弹出确认框点确定 → 自动执行 `pip uninstall pyautogui pillow`
3. USB-AI 完全正常使用，仅自动化功能不可用

### 离线安装
将 `pyautogui` 和 `pillow` 的 whl 文件放入 `runtime/auto-deps/` 目录，安装时将优先使用本地包，无需联网。

## 十一、常见问题

**Q: 搜索超时**
A: 网络慢时正常。直达数据（黄金/汇率/彩票）1s 内返回。减少搜索范围或手动搜索。

**Q: DuckDuckGo 搜索失败**
A: DDG API 已永久返回非 JSON（被云防护拦截），系统自动熔断跳过，不影响 Bing/Sogou。Bing+搜狗覆盖 99% 搜索需求。

**Q: AI 说"无法联网"**
A: 刷新页面（Ctrl+F5）。确保服务器已重启加载最新代码。查看调试日志确认 [RAG] ctx= 有数据。

**Q: 端口被占用**
A: 运行 `taskkill /F /IM python.exe` 后重启。或修改 server.py 中 PORT 变量。

**Q: 直连模式能用吗**
A: 可以，但无联网搜索和桌面自动化。双击 `启动.bat` 或直接打开 index.html。

**Q: AI 提取数据准不准**
A: AI 提取（`ai_extract`）先用 DeepSeek 做结构化提取，失败自动回落正则。测试覆盖彩票/体育/金融/表格/通用文本。同一查询 10 分钟内缓存命中不重复调用 AI。

**Q: 本地工具 `/api/tool` 安全吗**
A: 三层防护：(1) 路径沙箱限定 `workspace/data/uploads`；(2) Shell 白名单 16 个命令 + 40+ 黑名单；(3) Shell 命令需 API Key，文件工具无需认证。全部操作记入 `data/tool_log.jsonl`。

**Q: 桌面自动化打勾显示请求失败**
A: 重启服务器后刷新页面（F5）即可。v4.5.6 后 Token 已持久化，后续重启不再失效。

**Q: 桌面自动化安装失败**
A: 自动清除了 Windows 系统代理环境变量。也可手动将 whl 文件放入 `runtime/auto-deps/` 离线安装。

**Q: 之前保存的设置不见了**
A: 设置存于浏览器 localStorage，清缓存或换浏览器会丢失。配置后点保存即可。
