# USB-AI 操作手册 v4.3

## 一、快速开始

### Windows
1. 解压到任意目录（含 U 盘）
2. 双击 `启动服务器模式.bat`
3. 浏览器自动打开 `http://localhost:8082`
4. 点击右上角 ⚙ → 填入 DeepSeek API Key → 保存

### 首次使用
- 注册 DeepSeek: https://platform.deepseek.com
- 获取 API Key（sk- 开头）
- 在设置中填入并保存

## 四、联网搜索

### 智能触发（推荐）
默认关闭。当问题包含实时关键词时**自动搜索**：
- 财经：黄金/白银/比特币/汇率/石油/期货
- 天气：气温/台风/雾霾
- 彩票：快乐8/大乐透/双色球
- 体育：比分/NBA/英超/欧冠
- 商品：价格/性价比/多少钱
- 汽车：买车/报价/配置/保养
- 手机：iPhone/华为/小米/回收
- 考试：高考/考研/雅思
- 新闻：新闻/热点/快讯
- AI：GPT/Claude/大模型

### 手动搜索
输入 `/search 关键词` → 显示搜索结果 → 继续追问 AI 会基于结果回答

### 强制开关
🌐 按钮打开 → 所有问题都搜索；关闭 → 仅实时关键词触发

## 七、本地模型（可选）

1. 下载安装 Ollama: https://ollama.com
2. 终端运行: `ollama pull deepseek-r1:1.5b`
3. 重启 USB-AI 服务器
4. 设置中选择 "R1 1.5B (本地)"

## 八、调试

设置中勾选"显示调试日志" → 左侧底部出现日志面板，显示：
- [ROUTE] 路由决策
- [SEARCH] 是否搜索
- [RAG] 搜索结果（ctx=字符数 src=来源数）
- [TOKEN] Token 估算
- [CACHE] 缓存命中

## 九、常见问题

**Q: 搜索超时**
A: 网络慢时正常。直达数据（黄金/汇率/彩票）1s 内返回。减少搜索范围或手动搜索。

**Q: AI 说"无法联网"**
A: 刷新页面（Ctrl+F5）。确保服务器已重启加载最新代码。查看调试日志确认 [RAG] ctx= 有数据。

**Q: 端口被占用**
A: 运行 `taskkill /F /IM python.exe` 后重启。或修改 server.py 中 PORT 变量。

**Q: 直连模式能用吗**
A: 可以，但无联网搜索。双击 `启动.bat` 或直接打开 index.html。
