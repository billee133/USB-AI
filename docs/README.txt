🚀 快速开始
-----------

【Windows】
  方式一 — 双击 "启动.bat"（直连模式，仅需浏览器）
  方式二 — 双击 "启动服务器模式.bat"（代理模式，支持联网搜索+直达数据）

【Linux / Mac】
  bash start.sh（服务器代理，推荐）

🔑 API Key
----------
访问 https://platform.deepseek.com 注册获取 sk- 开头的密钥
在页面点击 ⚙ 设置，填入 Key 后保存


🌐 联网搜索
-----------
默认不搜索。以下关键词自动触发：
  黄金/白银/比特币/汇率/石油/天气/彩票/比分/新闻/价格/考试/AI资讯...
手动搜索：输入 /search 关键词


💻 本地模型
-----------
安装 Ollama → ollama pull deepseek-r1:1.5b → 设置中选择即可
