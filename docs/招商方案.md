## 第3页：核心功能

### 🤖 AI 对话
- 流式打字机输出 + Markdown/KaTeX/代码高亮
- 支持 DeepSeek V4 Pro / V4 / V3 / R1
- 本地 Ollama 模型一键切换
- 对话历史自动保存，刷新恢复

### 🔍 智能搜索
- 默认不搜，实时关键词自动触发
- AI 自主决定何时搜索 [SEARCH:关键词]
- 并行执行：直达API(1s) + 网页搜索(10s)，谁先到先用

### 📊 实时数据
- 贵金属：黄金 $4,190/oz、白银 $65.95/oz（实时）
- 加密货币：BTC $65,052、ETH $1,754（Binance 实时）
- 汇率：USD/CNY 6.789（实时）
- 原油：WTI/布伦特（抓取）
- 彩票：快乐8/大乐透/双色球 30+ 彩种（huiniao API）
- 天气：全国城市实时
- 汽车：报价/配置/保养/保险/回收
- 手机：价格/配置/回收

### ⚡ Token 优化
- 零Token模式：问候/时间/计算本地处理
- 响应缓存：24h TTL，重复查询零成本
- 历史压缩：超8轮自动摘要，节省60%
- 按需Prompt：场景动态加载

## 第4页：技术架构

```
前端 (355行 JS)          服务端 (2099行 Python)
     │                         │
     ├─ 零Token处理器          ├─ RAG管道 (搜索→抓取→AI提取)
     ├─ 智能搜索门             ├─ 15+ 直达数据API
     ├─ 响应缓存(24h TTL)      ├─ 多引擎搜索(Bing+Sogou+DDG)
     ├─ 历史压缩(8轮+摘要)     ├─ AI结构化提取
     ├─ Token估算器            ├─ 并行搜索调度(4线程)
     └─ AI工具调用             └─ SQLite本地存储
```

- 纯 Python 标准库，无第三方依赖
- 前端纯 JavaScript + CDN 加速
- SQLite 本地存储，零配置
